[L0034] Utilisation de vos données
[L0035] Google utilise vos données comme décrit dans ses Règles de confidentialité       pour :
[L0036] fournir ses services ;
[L0037] maintenir et améliorer ses services ;
[L0038] développer de nouveaux services ;
[L0039] personnaliser ses services ;
[L0040] adapter ses services ;
[L0041] communiquer avec vous ;
[L0042] mesurer les performances ;
[L0043] protéger Google, ses utilisateurs et le public.
[L0044] Ces utilisations s'étendent aux modèles d'IA générative et aux autres technologies de machine learning qui alimentent nos services.
[L0045] Des réviseurs humains (y compris des réviseurs spécialisés de nos fournisseurs de services) examinent certaines des données que nous collectons à ces fins.
[L0046] Veuillez ne pas saisir d'informations confidentielles que vous ne souhaiteriez pas qu'un réviseur consulte ou que Google utilise pour améliorer ses services, y compris ses technologies de machine learning.

[L0052] Les applications Gemini fonctionnent avec les Applis connectées, y compris les applications Google et des services tiers.
[L0053] Gemini enregistre et utilise les informations provenant des Applis connectées conformément à cet avis, y compris pour fournir et améliorer les applications Gemini.

[L0055] Si vous utilisez les applications Gemini pour interagir avec des services tiers, ceux-ci traitent vos données conformément à leurs propres règles de confidentialité.
[L0056] Fonctionnalités audio
[L0057] Les applications Gemini peuvent s'activer alors que ce n'était pas votre intention, par exemple si elles détectent un bruit ressemblant à "Hey Google" ou si vous les activez involontairement par le toucher.
[L0058] Si les applications Gemini répondent, cette interaction sera traitée comme une activation intentionnelle.
[L0059] Selon vos paramètres, ces données sont utilisées pour améliorer les services Google.
[L0060] Ces services incluent les modèles Gemini , d'autres modèles d'IA générative qui alimentent les applications Gemini ainsi que des technologies qui contribuent à réduire les activations involontaires.
[L0061] Canvas
[L0062] Lorsque vous interagissez avec une application Canvas générée par un utilisateur, le créateur de l'application peut voir toutes les données que vous partagez et les enregistrer où il le souhaite.
[L0063] Google fournit un espace de stockage dans Firebase pour les données des applications Canvas, mais le créateur de l'application peut également enregistrer vos données ailleurs.
[L0064] Toute personne disposant du lien public peut également consulter et modifier les données enregistrées avec l'application.
[L0065] Soyez prudent et ne partagez des données qu'avec les applications Canvas auxquelles vous faites confiance.

[L0081] Combien de temps nous conservons vos données
[L0082] Nous conservons vos données pendant des périodes plus ou moins longues en fonction de leur type et de l'utilisation que nous en faisons, comme expliqué dans les Règles de confidentialité de Google et dans la section Comment les informations collectées par Google sont-elles conservées .
[L0083] Vous pouvez modifier le paramètre de suppression automatique de vos données dans l'Activité dans les applications Gemini. Il est défini sur 18 mois par défaut, mais vous pouvez choisir une période de 3 mois, 36 mois ou illimitée. Vous pouvez aussi supprimer manuellement vos discussions dans les applications Gemini à tout moment.
[L0084] Les discussions examinées par des réviseurs humains (ainsi que les données associées comme la langue, le type d'appareil, les informations de localisation et vos commentaires) ne sont pas supprimées lorsque vous supprimez votre activité. Elles sont conservées pendant trois ans maximum.
[L0085] Nous conservons certaines données jusqu'à ce que vous les supprimiez de votre compte Google (par exemple, les informations sur la fréquence à laquelle vous utilisez les applications Gemini).
[L0086] Nous pouvons conserver certaines données plus longtemps lorsque cela est nécessaire pour des raisons légitimes d'ordre commercial ou juridique (par exemple pour des questions de sécurité ou d'archivage de données financières, ou pour prévenir la fraude et les abus).
[L0087] Découvrez-en plus sur les périodes de conservation des données            de Google, ainsi que sur les délais nécessaires à la suppression de vos informations.

[L0262] Comment les données de localisation sont-elles obtenues et utilisées par les applications Gemini ?
[L0263] Les applications Gemini peuvent utiliser différentes sources pour les données de localisation.
[L0264] Pour vous fournir des réponses utiles à vos prompts (comme "Quel temps fait-il ?"), les applications Gemini utilisent par défaut la zone géographique générale de votre adresse IP, ou l'adresse professionnelle ou personnelle figurant dans votre compte Google.

[L0266] Par exemple, les applications Gemini peuvent utiliser votre position exacte pour répondre plus précisément à des prompts comme "Où se trouve le café le plus proche ?".

[L0271] Le service Google qui reçoit vos données de localisation les utilise conformément aux Règles de confidentialité de Google        .

[L0273] Comment les données de localisation sont-elles stockées ?
[L0274] La position exacte traitée avec votre autorisation à partir de votre appareil ou navigateur n'est pas stockée dans votre Activité dans les applications Gemini.
[L0275] Les données de localisation traitées par les applications Gemini sont réduites à une zone géographique générale avant d'être stockées dans l'Activité dans les applications Gemini. Une zone géographique générale a une surface supérieure à 3 km² et compte au moins 1 000 utilisateurs. La zone associée à vos prompts ne permet donc pas de vous identifier, ce qui contribue à protéger votre confidentialité. En pratique, cela signifie qu'une zone géographique générale est habituellement bien plus vaste que 3 km² en dehors des villes.

[L0446] Gems de Google Labs
[L0447] Qu'advient-il de mes données lorsque j'interagis avec des Gems Google Labs ?
[L0448] Vous pouvez désormais accéder aux Gems de Google Labs          dans les applications Gemini ou sur opal.google .
[L0449] Les Gems de Google Labs       sont des workflows personnalisés regroupés dans des mini-applis d'IA que vous pouvez créer, modifier et partager.
[L0450] Lorsque vous interagissez avec des Gems de Google Labs, vous utilisez Opal, une expérience Google Labs.
[L0451] Important : Opal, une expérience Google Labs, ne fait pas partie des applications Gemini.
[L0452] Cela signifie que les données de cette expérience n'apparaissent pas dans votre activité dans les applications Gemini et ne sont pas contrôlées par le paramètre d'application connectée Workspace de Gemini.
[L0453] L'Avis de confidentialité des applications Gemini ne s'applique pas, et vos données Opal, y compris les commentaires que vous fournissez, sont traitées conformément aux Règles de confidentialité de Google    .
[L0454] Comment les données sont-elles stockées ?
[L0455] Les Gems de Google Labs      que vous créez, vos interactions avec eux et les contenus multimédias (audio, images, documents) que vous importez sont stockés dans le dossier "Opal" de votre Google Drive .
[L0456] Comment contrôler et gérer vos données, y compris supprimer vos Gems de Google Labs Vous pouvez supprimer des Gems de Google Labs dans le gestionnaire de Gems           de l'application Web Gemini ou dans le dossier "Opal" de votre Google Drive    .
[L0457] L'historique des interactions associées au Gem de Google Labs sera également supprimé.

[L0466] Canvas
[L0467] Qu'advient-il de mes données lorsque j'interagis avec des applications Web Canvas ?
[L0468] Lorsque vous interagissez avec des applications Canvas générées par des utilisateurs, les créateurs de ces applications peuvent voir toutes les données que vous ajoutez.
[L0469] Comment les données sont-elles stockées ?
[L0470] Google permet aux créateurs d'applications Canvas de stocker de manière permanente les données des utilisateurs dans Firebase.
[L0471] Les créateurs d'applications peuvent choisir cette option, mais ils peuvent également décider de stocker les données d'interaction vous concernant à d'autres endroits (ou encore de ne pas les stocker du tout).
[L0472] Comment contrôler et gérer vos données ?
[L0473] Soyez prudent et interagissez uniquement avec des applis auxquelles vous faites confiance.
[L0474] Une fois que vous avez partagé des données avec une application Canvas, vous ne pouvez pas les supprimer.
[L0475] Cependant, le créateur de l'appli peut le faire.
